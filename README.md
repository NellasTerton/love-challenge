# 💍 Sven & Tamilla Love Challenge (Telegram Mini App)

Интерактивное 30-дневное приложение-челлендж для двоих, разработанное в формате Telegram Web App (TWA). Проект создан в качестве уникального свадебного подарка, объединяющего геймификацию, трекинг привычек и систему вознаграждений.

[**Live Demo (Frontend)**](https://lovechallengesventamilla.netlify.app)

## ⚙️ Архитектура и Стек

**Frontend (Google Cloud Run):**
* **React + TypeScript** — логика приложения, управление состояниями (`localStorage`).
* **Tailwind CSS** — стилизация, адаптивный мобильный интерфейс.
* **Framer Motion** — сложная UI-анимация (3D-перевороты карточек, механика "веера" (deck) заданий).

**Backend (Amvera Cloud):**
* **Python 3.12**
* **Aiogram 3.x** — асинхронный Telegram-бот, обработка коллбеков, отправка пушей.
* **APScheduler** — планировщик для ежедневной рассылки заданий (Cron).
* **Aiohttp** — обработка сетевых запросов.
* *База данных:* Локальная JSON-структура (`users.json`, `state.json`) для хранения очков и ID пользователей.

## 🚀 Основной функционал

1. **Ежедневные квесты (Cron Jobs):** Каждое утро (10:00) бэкенд триггерит смену игрового дня и отправляет партнерам пуш-уведомления со свежей ссылкой в Mini App.
2. **Бесшовная авторизация и анти-кэш:** Telegram ID, текущий игровой день и актуальные баллы прокидываются в React-приложение напрямую через зашифрованные URL-параметры. Внедрен timestamp-антикэш для обхода агрессивного кэширования Telegram WebView.
3. **Механика «Руки» (Веер заданий):** Пользователь может принять задание на сегодня или отложить его, формируя пул активных квестов ("в руке").
4. **Система верификации (P2P):** 
   * Выполнение задания триггерит скрытый HTTP-запрос к Telegram API без закрытия Web App.
   * Партнер получает пуш с кнопками (Confirm / Reject) в чате с ботом.
   * При подтверждении баллы зачисляются на счет, а статус в календаре обновляется.
5. **Магазин наград:** На 30-й день разблокируется система обмена заработанных очков (1-3 за таск) на реальные призы. Покупка также сопровождается P2P-уведомлением партнеру.
6. **Система «Джокеров»:** Ограниченная возможность (3 на игру) заменить карточку дня на альтернативную с потерей потенциальных баллов.

## 📂 Структура проекта
* `/src` — исходный код React-приложения (включая `data.ts` с конфигурацией заданий и наград).
* `/telegram-bot` — контроллер Telegram API и планировщик на Python.
